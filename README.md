# AI-Assistant-with-TCP-Networking

This project is a basic AI assistant implemented using Python socket programming (TCP). It supports multiple commands like translation, calculator, and can be extended to include features like emailing.

Files

server.py: The main server handling multiple clients.
client.py: The client script users run to interact with the assistant.
How to Run

Run server.py first.
Then run client.py from one or more terminals.
